# Healthcare Cloud Data Platform

## Project Overview

This project is an end-to-end healthcare data engineering platform designed to process hospital operational data and produce analytics-ready datasets.

It demonstrates practical skills in ETL, AWS S3, Snowflake, SQL, data quality, orchestration, logging, and healthcare analytics.

## Business Problem

Hospitals need reliable data to monitor patient flow, bed occupancy, surgery cancellations, and ward performance. This platform converts raw hospital data into trusted reporting datasets.

## Tech Stack

- Python
- SQL
- AWS S3
- Snowflake
- PostgreSQL
- Apache Airflow
- Docker
- GitHub
- Data Quality Checks
- Power BI-ready Analytics Views
