# Analytics Layer

## Purpose

The analytics layer provides SQL views for hospital operational reporting.

## Views Created

### vw_avg_length_of_stay_by_ward
Calculates average length of stay and total admissions per ward.

### vw_bed_occupancy_by_ward
Shows daily bed occupancy percentage by ward.

### vw_theatre_cancellation_rate
Calculates cancellation percentage by surgery date and specialty.

### vw_hospital_operations_summary
Combines admission and bed occupancy metrics into a management summary.

## Business Value

These views support healthcare decision-making around patient flow, bed capacity, surgery performance, and ward-level operational pressure.
